Final Challenge

In this final challenge you will be creating a little web application.
Which tools you are using and how you design your webapp is completely up to you.
So dont hesitate and feel free to let your programmer heart take over :).

Prerequisite:

For this web application you should create a new folder within this project.
If you want you can use any js framework you feel comfortable to implement this task but its not required to use any.
The end result should be a finished webapplication. Your created index.html should be the entry point to your application.

Task Description:

Your job is to implement a web application which displays a list of todos.
Those todos should be loaded from following endpoint https://jsonplaceholder.typicode.com/todos
This Endpoint gives you a total of 200 Todo objects.
When your index.html is loaded the first 10 Todos should be displayed automatically.
There should be a element which enables the user to display more Todos (the next 10 Objects).
Every todo you get from this endpoint has an attribute "completed:", if this attribute is true this todo element should be marked as done
You could for example color completed Todos green or show an checkmark within those elements. Be creative ^^.
TIP: you can load all 200 todos on load and fill in the first 10. Its not a requirement to load only 10 todos at a time.

How you display the list of todos is completely up to you. It should definitely be user friendly as well as good designed.
Try to embed all common SEO (Search Engine Optimization) features you are familiar with.
Don't forget to comment your code.
