REST Api Challenge

In this challenge you will be working with a REST Api.
This is a free api which gives you a list of Todos, Users, posts etc.
In the following tasks you have to work with the api and manipulate its response.
In order to fetch the response of the JSONPlaceholder api api you should be using JavaScript.

TIPS:
I recommend you to create a separate js file which handles the api calls and all event handlers needed for the tasks.
There are countless js libraries which enable you to easily fetch data from an api.

When clicking on the button "GET" a request to the api should be made. The ouput should be displayed in the console.
All TODOs should be loaded.
TIP:
The Todos can be loaded under /todos (see here under Ressources )
