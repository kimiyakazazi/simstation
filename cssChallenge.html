CSS challenges

Within these CSS Challenges your task is to create a similar shape you see on the left hand side. Each specific task tells you where you can find the file you are supposed to implement your solution in. The first solution is already done for you, so you get the hang of it. There will be no need to use js for these css Challanges everything should be done with css and html. Every tasks has a "Colors to Use Section" there you'll find all colors needed to create your shape. Have fun and be creative :)
