CSS challenges

In the following 4 challenges/tasks you will work on some CSS challenges and a little bit of JavaScript.
At the end of the exercise, you will code a small web application that connects to a free API.
A separate description for each task can be found.
If you have any questions, please feel free to contact me at any time: p.kogler@simstation.com

A few rules you should follow:
I want you to use Git as your versioning system. I recommend that you use GitHub
Simply upload this folder to a new repository of your choice. When you completed a task, commit the changed or created files to your repository
Please make sure to commit regularly and meaningfully.
When you have completed all tasks, send me the link to your repository and I will track your progress and commits.
