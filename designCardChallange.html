Design Challenge

In this challenge you are supposed to design a component based on a (text) description as well as a hand drawn sketch. The hand drawn sketch is only a rough visualization of what we want from you. So dont hesitate to adjust your design and component as you see fit.
You can use HTML, CSS and JS. If you want to use a design framework other than bootstrap feel free to import it within this file. Your component should be implemented within the below div with the id "custom component"
We want a new component for our website.
This component will be used as a teaser card for various articles.
The Component should be able to:
  • display a header image previewing the article
  • display the article Title
  • display the article subTitle
  • display an article short description (max. 250 character)
The Short description of the Article should be collapsable. On default the cards description is collapsed.
